Title: Animated E-commerce Website

Overview:

This project is an interactive e-commerce website prototype for a shoe store, featuring smooth animations and a modern design. It showcases the latest Nike shoes with vibrant colors and detailed product information.

Role:

I designed and developed the entire website, focusing on creating an appealing user interface with seamless animations to enhance user experience.

Tools and Technologies:

HTML, CSS, JavaScript, AOS (Animate On Scroll Library), Boxicons, Remixicon

Challenges:

Implementing smooth animations and ensuring cross-browser compatibility were the main challenges.

Outcome:

The final product is a responsive and dynamic website that captures users' attention and provides a modern shopping experience.

Prime Video Clone Project

Overview:

The Prime Video Clone project is a front-end web development project designed to mimic the user interface of Amazon Prime Video. This project showcases a visually appealing homepage that features popular shows and movies, providing an engaging user experience.

Features

Interactive Navigation: A navigation bar with links to different sections such as Home, Store, Live TV, and Categories.

Technologies Used

HTML5: Used for structuring the content and elements on the page.

CSS3: Used for styling the website, including layout, colors, and fonts. The project also utilizes CSS for animations and transitions to enhance user experience.
